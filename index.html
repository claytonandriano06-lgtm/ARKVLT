This is not a SaaS platform. It is a personal automation tool for one creator, one account.

Privacy Policy

ARKVLT Manager does not collect, store, or share any personal data from TikTok users. Authentication is handled entirely through TikTok's official OAuth 2.0 system. The only data accessed is the creator's own account profile (open ID, display name) for authentication verification purposes.

No third-party data collection. No analytics. No cookies beyond session management. Access tokens are stored locally on the creator's own device only and are never transmitted to any third-party server.
